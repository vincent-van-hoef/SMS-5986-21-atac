Job summary

  • Enrichment method: GSEA
  • Organism: hsapiens
  • Enrichment Categories: pathway_KEGG
  • Interesting list: a R object. ID type: genesymbol
  • The interesting list contains 12154 user IDs in which 10829 user IDs are unambiguously mapped to 10829 unique entrezgene IDs and 1325 user IDs can not be mapped to any entrezgene ID.
  • The GO Slim summary are based upon the 10829 unique entrezgene IDs.
  • Among the 10829 unique entrezgene IDs, 3883 IDs are annotated to the selected functional categories, which are used for the enrichment analysis.

Parameters for the enrichment analysis:

  • Minimum number of IDs in the category: 10
  • Maximum number of IDs in the category: 500
  • Significance Level: Top 20
  • Number of permutation: 1000

Based on the above parameters, 20 positive related categories and 20 negative related categories are identified as enriched categories , in which 20 most significant categories and represetatives in the reduced sets are shown in this report. All significant categories can be downloaded from the 'Result Download' link.

GO Slim summary for the user uploaded IDs

Each Biological Process, Cellular Component and Molecular Function category is represented by a red, blue and green bar, repectively.

The height of the bar represents the number of IDs in the user list and also in the category.
